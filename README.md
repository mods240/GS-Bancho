# ⛽ ガス番長

ルート上の最安ガソリンスタンドを素早く探すためのWebアプリです。  
**GitHub Pages で無料公開 → スマホのホーム画面に追加** して使います。

---

## 📱 アプリの機能

- **現在地を自動取得**（GitHub Pages は HTTPS なので動作します）
- **目的地を入力 → gogo.gs でルート沿いの最安値を検索**
- **スタンドが決まったら → Google マップでナビ起動**
- **給油メモ**（価格・給油量・合計金額をスマホに記録）

---

## 🚀 公開手順（コピペだけで完了）

### ステップ 1：GitHub アカウントを作る

1. [https://github.com](https://github.com) を開く
2. 「Sign up」→ メールアドレス・パスワード・ユーザー名を入力
3. メール認証を完了する

---

### ステップ 2：リポジトリを作る

1. GitHub にログインしたら、右上の「＋」→「New repository」をクリック
2. 設定：
   - **Repository name**: `gas-bancho`（なんでも可）
   - **Public** を選択（Pagesを使うため）
   - 「Create repository」をクリック

---

### ステップ 3：ファイルをアップロードする

1. 作成したリポジトリのページで「uploading an existing file」をクリック
2. 以下の **4ファイル** をドラッグ＆ドロップ：
   ```
   index.html
   style.css
   app.js
   manifest.json
   icon.svg
   ```
3. 下の「Commit changes」をクリック

---

### ステップ 4：GitHub Pages をオンにする

1. リポジトリの「**Settings**」タブをクリック
2. 左メニュー「**Pages**」をクリック
3. Branch を「**main**」に変更 → 「Save」
4. 数分後、以下の URL でアクセスできます：

```
https://あなたのユーザー名.github.io/gas-bancho/
```

---

### ステップ 5：スマホのホーム画面に追加

**iPhone（Safari）**
1. Safariで上記URLを開く
2. 下の「共有」ボタン → 「ホーム画面に追加」

**Android（Chrome）**
1. Chromeで上記URLを開く
2. 右上メニュー「…」→「ホーム画面に追加」

---

## 📖 使い方

1. アプリを開くと**現在地を自動取得**します（許可してください）
2. 「目的地」に行き先を入力（例：大阪城、名古屋駅）
3. 「ルート上の最安値を探す」をタップ → **gogo.gs が開く**
4. gogo.gs のマップでルート沿いの安いスタンドを確認
5. スタンドが決まったらアプリに戻り、スタンド名を入力 → **ナビ開始**

---
